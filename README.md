# ZeeQue Admission Portal

A premium, high-performance landing page for ZeeQue Preschool admissions built with Next.js 14, Tailwind CSS, and Framer Motion.

## Features

- **Next.js 14 App Router:** Modern React architecture.
- **Tailwind CSS v3+:** Custom luxury theme configuration extending default utilities.
- **Framer Motion:** Smooth entrances, scroll animations, and interactive elements.
- **React Hook Form + Zod:** Robust client-side validation for the multi-step enquiry form.
- **Server Actions:** Mock API endpoint for form submissions.
- **Responsive Design:** Optimized for mobile, tablet, and desktop screens with a customized sticky header.
- **Accessibility:** Semantic HTML and ARIA optimizations.

## Getting Started

1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Run Development Server:**
   ```bash
   npm run dev
   ```

3. **Open:** Navigate to [http://localhost:3000](http://localhost:3000)

## Project Structure

- `app/` - Next.js App Router root, global styles, server actions.
- `components/` - High-quality dynamic React components built with Framer Motion.
- `public/` - Static assets.
